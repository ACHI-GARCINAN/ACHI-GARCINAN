 # נוסחאות התלמוד

**נוסחאות התלמוד** הינה תוכנה המיועדת להצגת שינויי נוסחאות בין מהדורות והוצאות שונות של דפי הגמרא.

## אודות הפרויקט
התוכנה מאפשרת עיון והשוואה בין עדי נוסח שונים בצורה נוחה וידידותית, תוך דגש על חוויית משתמש חלקה.
